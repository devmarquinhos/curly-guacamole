### Instalação

Você vai precisar do [NodeJS](https://nodejs.org) instalado na sua máquina para rodar essa aplicação.

```Git
git clone https://github.com/devmarquinhos/curly-guacamole
$ cd curso-react
$ npm install
$ npm run dev
```
